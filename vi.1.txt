.Sh NAME
.Nm nextvi
.Nd A small vi/ex editor for editing bidirectional UTF-8 text
.
.Sh SYNOPSIS
.Nm vi
.Op Fl emsv
.Op Ar
.
.Sh DESCRIPTION
.Nm nextvi
is a
.Xr vi 1 Ns -like
text editor based on Ali Gholami Rudi's
.Xr neatvi 1 Ns .
It supports UTF-8 and bidirectional text.
.
.Sh OPTIONS
.Bl -tag -width Ds
.
.It Fl e
Enter Ex mode on startup
.
.It Fl m
Disable initial file read message
.
.It Fl s
When combined with -e enters limited Ex mode
.
.It Fl v
Enter visual mode on startup. (Default)
.
.Sh VI NORMAL
.Bl -tag -width Ds
.It Cm arg-+ arg-\en arg-j
Move the cursor arg lines down
.It Cm arg-- arg-k
Move the cursor arg lines up
.It Cm ^
Move the cursor to the start of the line after indentation
.It Cm '
Goto a line mark
.It Cm gg
Goto the first line in the buffer
.It Cm G
Goto the last line in the buffer
.It Cm H
Goto the highest line of the screen
.It Cm L
Goto the lowest line of the screen
.It Cm M
Goto the middle line of the screen
.It Cm arg-,
Repeat last f or F or t or F command backward arg times
.It Cm arg-;
Repeat last f or F or t or F command forward arg times
.It Cm arg-h
Move the cursor arg cols left
.It Cm arg-l
Move the cursor arg cols right
.It Cm f
Move the cursor to the character found forward
.It Cm F
Move the cursor to the character found backward
.It Cm t
Move the cursor until the character found forward
.It Cm T
Move the cursor until the character found backward
.It Cm arg-B
Move the cursor to the end of the word skipping punctuation backward arg times
.It Cm arg-E
Move the cursor to the end of the word skipping punctuation forward arg times
.It Cm arg-b
Move the cursor to the end of the word backward arg times
.It Cm arg-e
Move the cursor to the end of the word forward arg times
.It Cm arg-W
Move the cursor to the start of the word skipping punctuation forward arg times
.It Cm arg-w
Move the cursor to the start of the word forward arg times
.It Cm {
Move the cursor to the next { region
.It Cm }
Move the cursor to the next } region
.It Cm [
Move the cursor to the next \en region up
.It Cm ]
Move the cursor to the next \en region down
.It Cm C-]
.It Cm C-p
.It Cm C-t
.It Cm 0 |
Move the cursor to the start of the line.
.It Cm $
Move the cursor to the end of the line.
.It Cm arg-|
Move the cursor to arg col
.It Cm arg-/
Search using regex down skipping arg matches
.It Cm arg-?
Search using regex up skipping arg matches
.It Cm arg-n
Repeat search down skipping arg matches
.It Cm arg-N
Repeat search up skipping arg matches
.It Cm C-a
Auto search word under the cursor, not centering and wrapping up/down direction
.It Cm arg-C-a
Auto search setting arg words from the cursor
.It Cm arg-Space
Move cursor arg character forward
.It Cm arg-C-h arg-Backspace
Move cursor arg character backward
.It Cm `
Goto a line mark with horizonal position
.It Cm %
Move the cursor to [](){} pair
.It Cm arg-%
Move the cursor to the arg % representation line of the file
.It Cm \\
.El
.
